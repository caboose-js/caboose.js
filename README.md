# Usage
<!-- usage -->
```sh-session
$ npm install -g caboose.js
$ caboose COMMAND
running command...
$ caboose (-v|--version|version)
caboose.js/1.0.0 win32-x64 node-v14.15.4
$ caboose --help [COMMAND]
USAGE
  $ caboose COMMAND
...
```
